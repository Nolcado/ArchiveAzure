**DOSSIER INTERNE - Niveau 3**

Agence du C.C.P.

Statut : Officier

Nature : Agence gouvernementale clandestine.

L'existence de la C.C.P. n'est reconnue par aucune administration officielle. Les rares documents la mentionnant sont classifiés ou fortement censurés.

  -Missions

Localiser et récupérer les individus classifié Code 6.
Kidnapper les individus classifié Code 5.
Surveiller les individus classifié Code 4
Interroger les individus classifié Code 3
Eliminer les individus classifié Code 2.
Enregistrer les individus classifié Code 1 dans la base de données.
Étudier, contenir et documenter les anomalies.
Assurer la confidentialité des incidents impliquant des phénomènes anormaux.
Confisquer ou détruire les preuves susceptibles d'être rendues publiques.

  -Les Codes

Code 1 : Individu soupçonné d'être en contact indirect avec une anomalie. Aucune preuve concluante.
Code 2 : Individu ayant été mis au courant des anomalies.
Code 3 : Individu ayant établi un contact volontaire avec une anomalie ou avec une organisation liée aux anomalies.
Code 4 : Individu présentant des capacités anormales sans exposition officiellement confirmée.
Code 5 : Individu ayant neutralisé ou éliminé une anomalie.
Code 6 : Individu ayant été exposé à une anomalie et présentant des altérations physiques, psychologiques ou comportementales.

  -Relations avec le Fabism

Les relations entre la C.C.P. et le culte du Fabism ne sont pas de ce niveau.

  -Note au personnel

Les classifications supérieures au Code 6 sont réservées au personnel de niveau d'accréditation 4 ou supérieur.
