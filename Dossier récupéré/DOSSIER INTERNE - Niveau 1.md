**DOSSIER INTERNE - Niveau 1**

Agence du C.C.A.



Statut : Recrue



Nature : Agence gouvernementale clandestine.



L'existence de la C.C.A. n'est reconnue par aucune administration officielle. Les rares documents la mentionnant sont classifiés ou fortement censurés.

L'existence de la C.C.A. demeure confidentielle. Les agents sont soumis à un devoir absolu de discrétion.



  -**Missions**



Identifier toute activité anormale signalée sur le territoire.

Signaler aux statuts supérieurs toute preuve susceptible d'être rendue publique.

Enregistrer les individus classifiés Code 1 dans la base de données.

Surveiller les individus classifiés Code 4.

Localiser les individus classifiés Code 6.



  -**Les Codes**



Code 1 : Individu soupçonné d'être entré en contact indirect avec une anomalie. Aucune preuve concluante.

Code 4 : Individu présentant des capacités anormales sans exposition officiellement confirmée.

Code 6 : Individu ayant été exposé à une anomalie.



  -**Procédure de confidentialité**



Tout incident impliquant une anomalie doit être signalé dans un délai maximal de 30 minutes.



La diffusion d'informations relatives aux anomalies en dehors du C.C.A. est strictement interdite. Vous serez alors classifié Code 2.




  -**Note au personnel**



Certaines informations ne sont accessibles qu'aux agents disposant d'une autorisation de Niveau II ou supérieure.

La consultation d'un document au-dessus de votre niveau d'autorisation constitue une violation du règlement interne.
